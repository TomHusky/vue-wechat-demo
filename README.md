## 一个模仿PC微信客户端的纯前端vue项目

> 项目地址
github:  https://github.com/TomHusky/vue-wechat-demo
gitee:  https://gitee.com/luowenjie98/vue-wechat-demo



目前已经完成的功能点有

- 登录
- 页面拖拽移动
- 好友聊天
- 群聊
- 消息通知提醒，免打扰，不显示聊天框
- 表情消息，图片消息，文件消息
- 通讯录
- 加好友，创建群聊
- 好友头像查看信息

## 项目启动


### 安装依赖
```
npm install
```

### 启动服务
```
npm run serve
```

### 构建输出
```
npm run build
```
